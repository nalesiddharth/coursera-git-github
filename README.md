- `.git/` - not included here. Push these files to a new GitHub repository and follow steps in the assignment instructions.

## How to run `simple-interest.sh`
Make the script executable and run it:

```bash
chmod +x simple-interest.sh
./simple-interest.sh --principal 1000 --rate 5 --time 2
```

Or interactive:

```bash
./simple-interest.sh
# then enter values when prompted
```
