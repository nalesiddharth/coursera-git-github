# Contributing

Thank you for taking the time to contribute to this demo assignment.

Suggested workflow:
1. Fork this repository on GitHub.
2. Clone your fork locally:
   `git clone https://github.com/<your-username>/demo-assignment.git`
3. Create a branch for your work:
   `git checkout -b bug-fix-typo`
4. Make edits (for example, change README.md).
5. Commit your changes:
   `git add .`
   `git commit -m "Fix: typo in README"`
6. Push the branch:
   `git push origin bug-fix-typo`
7. Create a Pull Request from `bug-fix-typo` into `main` in the GitHub UI.
8. Merge the PR (or accept it) to simulate the required merge for the assignment.
9. To create the `bug-fix-revert` branch and revert, create branch and then revert the relevant commit.

This file also documents how to run the simple-interest script and test it locally.
